## FASE 5: VIDEO DEMO (30 minuti)

### Cosa Mostrare (IN ORDINE)

**1. Home page (2 min)**
- Mostra come client vede il salone
- Clicca su "PRENOTA ORA"
- Evidenzia: niente account, niente registrazione

**2. Booking Flow (5 min)**
- Scegli data (mostra calendario)
- Scegli servizio (mostra lista completa personalizzabile)
- Mostra blocco orari automatico per durata servizio
- Inserisci dati client
- Mostra riepilogo prenotazione
- Conferma e mostra token link

**3. Client View (3 min)**
- Apri link token (prenotazione.html)
- Mostra dettagli prenotazione
- Mostra opzioni: modifica, annulla, lascia review, segnala problema
- Spiega: cliente può gestire tutto da link privato

**4. Admin Dashboard (10 min)**
- Login parrucchiere (mostra credenziali di default, spiega cambierà)
- **Dashboard**: Statistiche oggi (prenotazioni, completate, incasso)
- **Calendario**: Mostra appuntamenti visualizzati
- **Prenotazioni**: Tabella filtrata, status (confirmed/completed/cancelled)
- **Clienti**: Lista clienti con storico prenotazioni
- **Servizi**: Mostra 4 servizi demo, spiega come aggiungerne altri
- **Impostazioni**: Mostra dove personalizzare nome salone, orari, info
- **Segnalazioni**: Mostra come tracciare problemi clienti

**5. Personalizzazione (5 min)**
- Vai a Impostazioni
- Mostra come cambia nome in tutta l'app
- Spiega: numero telefono, email, indirizzo, orari
- Mostra anteprima come cambiano in home page
- Domanda: "Vi piacerebbe il vostro logo al posto di PR?"

**6. Domande Critiche & Risposte**

| Domanda | Risposta |
|---------|----------|
| "Quanto costa?" | "Zero commissioni. Una sola fee di setup per personalizzazione." |
| "Chi ha i dati?" | "Voi. Nel vostro cloud privato, voi siete i proprietari." |
| "Se fallisce il sistema?" | "I dati rimangono vostri. Facciamo backup settimanali." |
| "Quanto tempo per setup?" | "30 minuti di call con me, poi è pronto." |
| "Clienti devono registrarsi?" | "No. Nome + telefono. Zero attrito." |
| "Integrazione con nostro sito?" | "Si, basta un link. Oppure dominio separato." |
| "Support tecnico?" | "Sono io, disponibile 6 giorni su 7 via WhatsApp/email." |
