## ⚡ Ultra-Quick Deployment (Copy-Paste)

### Step 1: Create GitHub Repo
1. Go to https://github.com/new
2. **Repository name**: `parrucchiere-rossi`
3. **Visibility**: Public
4. Click **"Create repository"**
5. **Copy the commands** GitHub shows you

### Step 2: Push Code to GitHub
```bash
cd C:\Users\Utente\Desktop\Artigiani
git remote add origin https://github.com/[YOUR_USERNAME]/parrucchiere-rossi.git
git branch -M main
git push -u origin main
```

### Step 3: Deploy to Vercel
1. Go to https://vercel.com/new
2. Click **"Sign Up"** → **"Continue with GitHub"**
3. Authorize Vercel
4. Select `parrucchiere-rossi` repository
5. Click **"Deploy"**

⏳ **Wait 1-2 minutes...**

✅ **LIVE!** Your URL: https://parrucchiere-rossi.vercel.app
