================================================================================
  PROBLEMI COMUNI
================================================================================

PROBLEMA: "Firebase enabled: false"
  SOLUZIONE:
    1. Ricaricare pagina (Ctrl+F5)
    2. Aspettare 5 secondi
    3. Se ancora false, controllare connessione internet

PROBLEMA: "Booking salvato" NON appare
  SOLUZIONE:
    1. Verificare che nessun errore rosso in console
    2. Ricaricare pagina (F5)
    3. Riprovare a fare prenotazione
    4. Se persiste, controllare Firebase status

PROBLEMA: Prenotazione non appare in dashboard
  SOLUZIONE:
    1. Aspettare 5 secondi (auto-refresh)
    2. Ricaricare dashboard (F5)
    3. Se ancora non appare, verificare console dashboard
    4. Controllare che prenotazione sia salvata in console cliente

PROBLEMA: Errore rosso in console
  SOLUZIONE:
    1. Leggere messaggio di errore
    2. Se "Cannot use import statement", ricaricare pagina
    3. Se "Firebase not defined", verificare firebase-config.js
    4. Se altro, contattare supporto con screenshot errore
